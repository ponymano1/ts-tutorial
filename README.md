# TS 教程 / TS Tutorial

## 目标

TypeScript 在 Web3 中应用非常广泛。然而，很多 Web2 行业的程序员主要工作语言是 Java、Golang、C++ 等。
本教程主要面向已经有较多编程经验、但没有 TypeScript 开发经验的人，帮助你在一两天内快速过一遍语法并进入实际项目。
不建议没有编程经验的读者将本教程作为入门编程的第一份学习资料。

## Goal

TypeScript is widely used in Web3. However, many developers from Web2 primarily work with languages such as Java, Golang, and C++.
This tutorial is designed for experienced programmers who are new to TypeScript, helping them quickly review the syntax in one or two days and move into real projects.
It is not intended as a first programming tutorial for people without prior coding experience.
